Windows Server 2022 上遇到的问题

安装MySQL和初始化data文件夹时报错：找不到MSVCP140.dll文件

解决方法：安装 Microsoft Visual C++ Redistributable 客户端库，点击前往下载。

输入net start mysql时报错：服务没有响应控制功能

解决方法：下载VCRUNTIME140_1.dll，解压后拷贝到MySQL的bin文件夹下。
